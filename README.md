## ENV VARS and their defaults
```
REST:

API_JWT_SECRET -- required
JAVALIN_PROMETHEUS_PORT 7080
JAVALIN_HTTP2_PORT 8080
JAVALIN_HTTPS2_PORT 8443
JAVALIN_HTTPS2_CERT_CLASSPATH /keystore.jks
JAVALIN_HTTPS2_CERT_PASSWORD password

Redis:

API_REDIS_URL redis://localhost

Rabbitmq:

API_RABBITMQ_USERNAME guest
API_RABBITMQ_PASSWORD guest
API_RABBITMQ_HOST localhost
API_RABBITMQ_PORT 5672

Database:

API_DATABASE_URL jdbc:postgresql://[::1]/postgres
API_DATABASE_USERNAME postgres
API_DATABASE_PASSWORD postgres
API_DATABASE_MAXPOOLSZE 100


Booleans:

API_ENABLE_ENDPOINTS true
API_ENABLE_CONSUMERS false
API_ENABLE_SCHEDULED false
API_ENABLE_MIGRATION false
API_QUIT_AFTER_MIGRATION false
```
